# 玩家数值总表

这份文档汇总当前玩家相关的主要数值、公式、成长、技能和装备加成。

建议用途：
- 以后调玩家数值时，先改这份文档，再同步代码。
- 当前游戏不会直接读取这个 Markdown 文件；它是玩家数值的总表和调参入口。

当前运行时数值来源：
- 基础生命、等级、攻击、升级成长、回血规则：`game/autoload/GameManager.gd`
- 玩家移动、攻击、受击、头盔冲撞、近战判定：`game/scripts/player/Player.gd`
- 冰刃投掷：`game/scripts/player/IceBlade.gd`
- 装备效果：`game/autoload/EquipmentManager.gd`
- 技能效果：`game/autoload/SkillManager.gd`

### 2. 运行时覆盖关系

这些值不是单纯看 `Player.gd` 导出值，而是运行时会被其它系统覆盖：

| 项目 | 实际来源 | 当前公式/规则 |
|---|---|---|
| 玩家最大生命 | `GameManager.player_max_hp` | `100 + (等级 - 1) * 5` |
| 玩家当前生命 | `GameManager.player_hp` | 由受伤、升级回血、击杀回血、Boss 命中回血、坠落伤害共同改变 |
| 玩家基础攻击 | `GameManager.player_attack` | 初始 15，每升一级固定 `+1` |
| 玩家实际近战基础伤害 | `Player.attack_damage` | 每次 `hp_changed` 时同步为 `GameManager.player_attack` |
| 玩家最高滑行速度 | `Player.max_ski_speed` | `600 + 雪板加成 + 平行式滑雪加成` |

## 二、生命、升级与回血

### 1. 等级成长

| 项目 | 当前规则 |
|---|---|
| 最大生命成长 | 每级固定 `+5` |
| 攻击成长 | 每级固定 `+1`（另有速度奖励，见近战伤害公式） |
| 升级回血 | 回复当前最大生命的 `10% + 5` |
| 等级上限 | 由进化次数决定：`10 / 20 / 30 / 40` |

最大生命公式：

```text
最大生命 = 100 + (等级 - 1) * 5
```

基础攻击公式：

```text
基础攻击 = 15 + (等级 - 1) * 1
```

### 2. 进化与等级上限

| 进化次数 | 当前等级上限 | 解锁来源 |
|---|---:|---|
| 0 | 10 | 初始状态 |
| 1 | 20 | 击败 Boss1 冰川雪豹 |
| 2 | 30 | 击败 Boss2 冻结守卫 |
| 3 | 40 | 击败 Boss3 雪崩巨人 |

### 3. 回血规则

| 场景 | 当前规则 |
|---|---|
| 升级时 | 回复最大生命 `10% + 5` |
| 击杀任意敌人时 | 回复最大生命 `10%` |
| 命中 Boss 时 | 回复最大生命 `3%`，向上取整，最低 1 点 |

升级回血公式：

```text
回复量 = int(最大生命 * 0.1) + 5
```

Boss 命中回血公式：

```text
回复量 = max(1, ceil(最大生命 * 0.03))
```

### 4. 坠落伤害

| 项目 | 当前规则 |
|---|---|
| 触发条件 | 单次空中垂直落差超过 `300` |
| 伤害公式 | `int((h - 300) / 10) + 10` |
| 伤害来源 | 记作 `坠落` |

说明：
- `h` 表示玩家从离地后空中最高点到落地位置的垂直高度差。
- 只有真正落地时才会结算；高度不超过 `300` 不扣血。

## 三、移动与速度

### 1. 基础移动参数

| 项目 | 当前值 | 备注 |
|---|---:|---|
| `walk_speed` | 190 | 主要用于空中横向控制上限，以及上坡步行上限计算 |
| `ski_accel` | 440 | 地面基础加速度 |
| `ski_friction` | 180 | 平地滑行减速度 |
| `air_friction` | 45 | 空中横向控制基础值 |
| `max_ski_speed` | 运行时重算 | 实际由公式决定，不直接看导出默认值 |
| `jump_force` | -540 | 向上的跳跃初速度 |
| `gravity` | 980 | 重力 |

### 2. 最高滑行速度公式

```text
最高滑行速度 = 600 + 雪板速度加成 + 平行式滑雪速度加成
```

当前加成值：

| 来源 | 加成 |
|---|---:|
| 无装备/无技能 | 0 |
| 升级雪板 | +150 |
| 平行式滑雪 | +120 |

常见组合：

| 组合 | 最高滑行速度 |
|---|---:|
| 无装备、无技能 | 600 |
| 只有升级雪板 | 700 |
| 只有平行式滑雪 | 720 |
| 升级雪板 + 平行式滑雪 | 800 |

### 3. 斜坡与刹车规则

关键判定阈值：

| 项目 | 当前值 |
|---|---:|
| 斜坡判定阈值 | `abs(floor_normal.x) > 0.08` |
| 平行式刹车可用条件 | 在地面、按住 `brake`、且当前地形是斜坡 |

上坡时的速度上限：

```text
上坡最高速度 = walk_speed * abs(floor_normal.y) * 0.55
```

下坡时会额外叠加：

```text
下坡加速 ≈ gravity * slope_x
```

如果玩家还持续按方向键，则还会附加：

```text
额外输入加速 = accel * 0.15
```

## 四、近战攻击

### 1. 基础攻击参数

| 项目 | 当前值 |
|---|---:|
| 基础攻击力 | 15 起步，随等级每级 `+1`，另加速度奖励 |
| 攻击冷却 | 0.55 秒 |
| 前向攻击距离基值 | 90 |
| 攻击纵向尺寸 | 40 |
| 攻击判定持续时间 | 0.15 秒 |

### 2. 近战伤害公式

```text
速度奖励 = int(abs(水平速度) / 50)
近战实际伤害 = round((基础攻击 + 速度奖励) * (1 + 雪镜伤害加成))
```

其中：
- 基础攻击来自 `GameManager.player_attack`
- 速度奖励取自攻击瞬间的 `velocity.x`，每 50 单位速度额外 +1 伤害
- 雪镜伤害加成来自 `EquipmentManager.attack_damage_bonus()`

当前常见伤害（Lv.1，速度奖励示例）：

| 状态 | 速度 0 | 速度 300 (+6) | 速度 600 (+12) |
|---|---:|---:|---:|
| 无雪镜 | 15 | 21 | 27 |
| 雪镜 Lv.1 | 17 | 23 | 30 |
| 雪镜 Lv.2 | 20 | 28 | 35 |

### 3. 近战判定框公式

前向攻击距离会吃雪镜距离加成：

```text
前向距离 = 90 * (1 + 雪镜距离加成)
后向距离 = 前向距离 * 0.1
总宽度 = 前向距离 + 后向距离
中心偏移 = (前向距离 - 后向距离) * 0.5
```

当前常见数值：

| 状态 | 前向距离 | 后向距离 | 总宽度 | 中心偏移 |
|---|---:|---:|---:|---:|
| 无雪镜 | 90.0 | 9.0 | 99.0 | 40.5 |
| 雪镜 Lv.1 | 99.0 | 9.9 | 108.9 | 44.55 |
| 雪镜 Lv.2 | 108.0 | 10.8 | 118.8 | 48.6 |

## 六、头盔冲撞伤害

### 1. 触发条件

头盔冲撞不会依赖物理刚体碰撞，而是用前方范围检测结算。

触发条件：
- 已装备头盔
- 当前水平速度绝对值 `>= 350`
- 敌人在玩家前方近距离范围内

### 2. 头盔冲撞参数

| 项目 | 当前值 |
|---|---:|
| 触发速度阈值 | 350 |
| 前向检测距离 | 56 |
| 纵向检测距离 | 52 |

冲撞伤害公式：

```text
冲撞伤害 = int(基础攻击 * 头盔冲撞倍率)
```

当前头盔倍率：

| 头盔等级 | 冲撞倍率 |
|---|---:|
| 未装备 | 不触发 |
| Lv.1 | 1.5 |

示例：

| 基础攻击 | 头盔状态 | 冲撞伤害 |
|---:|---|---:|
| 20 | 无头盔 | 不触发 |
| 20 | 头盔 Lv.1 | 30 |

头盔可以减少 30% 的碰撞 / 冲撞伤害，并让受击时的横向击退速度减半。
