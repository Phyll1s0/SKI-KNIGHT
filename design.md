# Skiknight 游戏设计大纲

## 一、游戏概述

- **类型**：2D 类银河战士恶魔城（Metroidvania）
- **引擎**：Godot 4.x
- **主题**：滑雪骑士在雪山迷宫世界中探索、打怪、成长
- **核心体验**：利用滑雪动量穿越复杂地图、砍杀敌人、解锁新区域

---

## 二、核心玩法机制

### 2.1 滑雪移动系统
- 角色在斜坡上会自动积累速度（动量系统）
- 平地/上坡时速度衰减，下坡时加速
- 跳跃：可借助速度跳得更远（动量影响跳跃距离）
- 特殊动作：急停、漂移转向

### 2.2 战斗系统
- **普通攻击**：用雪杖/刀刃近战斩击
- **冲撞攻击**：高速滑行时撞击敌人造成额外伤害
- **技能攻击**（随成长解锁）：
  - 旋转斩（360° 雪刃）
  - 雪崩踩踏（跳起后下落）
  - 冰刃投掷（远程），雪球枪
- 受击有硬直，高速冲撞可打飞敌人

### 2.3 成长/升级系统
- 击败敌人获得经验，累计经验升级
- **每升一级**：最大血量 +10，攻击力 +1，并恢复 30% 最大血量
- **击杀敌人**：额外恢复 10% 最大血量
- **进化门槛**（等级上限锁）：
  | 等级上限 | 解锁条件 | 获得技能 |
  |---------|---------|--------|
  | 10 级锁 | 击败 Boss 1 冰川雪豹 → 进化 #1 | 平行式滑雪 |
  | 20 级锁 | 击败 Boss 2 冻结守卫 → 进化 #2 | 后空翻（二段跳）|
  | 30 级锁 | 击败 Boss 3 雪崩巨人 → 进化 #3 | 卡宾 |
  | 40 级上限 | 最终等级 | — |
  

### 2.4 Metroidvania 地图系统
- 大地图由多个互联区域组成
- 部分路径需要特定能力才能通过（如：冰壁需要火焰技能、高台需要二段跳）
- 解锁新能力 → 可回溯旧区域发现隐藏内容
- 设置地图标记和传送点


### 2.5装备系统
- 装备系统，最开始普通雪板，没头盔，后面可以通过**花金币**在商人处直接购买头盔（增加冲击伤害）
- 装备雪镜：增加视力还有攻击距离，二次升级雪镜可增加镜片提升照明能力（Boss2击败后获得升级零件，仍需花金币完成升级）
- 装备升级雪板：花金币在暴风雪高地工坊处升级，不升级雪板进不去第四部分地图
- 装备高级雪服：需要先从 Boss 1（冰川雪豹）处获得**雪服碎片**，再花金币在铁匠处合成；不穿雪服无法进入第三部分地图


### 2.6 技能机制 
- **平行式滑雪**（进化 #1，击败 Boss 1 冰川雪豹）：更快刹车、速度上限提升、可通行更陡坡度；进入暴风雪高地的必要条件
- **后空翻**（进化 #2，击败 Boss 2 冻结守卫）：空中二段跳能力，以 85% 第一跳力度起跳
- **卡宾**（进化 #3，击败 Boss 3 雪崩巨人）：移动更灵活，可走复杂路线，是进入雪山顶峰的必要条件


### 升级系统的提升

每升级一次：玩家 **最大血量 +10、攻击力 +1**，并恢复 **30% 最大血量**。
击杀任意敌人后额外恢复 **10% 最大血量**。

等级上限与进化门槛：
- **10 级上限** → 击败 Boss 1 冰川雪豹完成进化 #1，解锁**平行式滑雪**，上限解除至 20 级
- **20 级上限** → 击败 Boss 2 冻结守卫完成进化 #2，解锁**后空翻（二段跳）**，上限解除至 30 级
- **30 级上限** → 击败 Boss 3 雪崩巨人完成进化 #3，解锁**卡宾**，上限解除至 40 级
- **40 级**：最终等级上限

后面三张地图（暴风雪高地 / 地下冰窟 / 雪山顶峰）的敌人数值随阶段线性提升，体现挑战梯度。


### 2.7 金币系统

**来源**
| 来源 | 掉落金币 |
|------|---------|
| 雪人兵（普通） | 5–15 枚 |
| 冰球炮台（远程） | 10–20 枚 |
| 铠甲冰熊（精英） | 30–50 枚 |
| Boss 1 冰川雪豹 | 80 枚固定 + 雪服碎片 |
| Boss 2 冻结守卫 | 120 枚固定 |
| 最终 Boss 雪山之王 | 200 枚固定 |
| 隐藏宝箱 | 20–60 枚随机 |

**用途**

| 消费项目 | 花费 | 前置条件 | 购买地点 |
|---------|------|---------|---------|
| 头盔 | 60 金币 | 无 | 冰川迷宫商人 |
| 高级雪服合成 | 100 金币 | 持有雪服碎片（Boss1掉落） | 冰川迷宫铁匠 |
| 雪板升级 | 150 金币 | 无 | 暴风雪高地工坊 |
| 雪镜二级升级 | 80 金币 | 持有雪镜升级零件（Boss2掉落） | 任意商人 |

**商人/铁匠 NPC**
- 以 `StaticBody2D` 实现，在进入交互范围时显示 `[F] 购买` 提示
- 开启购买 UI（Panel）后可选择购买项目，不满足前置条件的选项变灰
- 交易完成后通过 `GameManager.add_gold(-cost)` 扣款，`EquipmentManager.unlock()` 解锁装备
- 金币数量由 `GameManager.gold: int` 持久化，`SaveSystem` 存档

## 三、地图/世界设定

| 区域 | 特色 | 解锁条件 |
|------|------|----------|
| 雪山入口 | 教学区，平缓斜坡，普通雪地地形 | 初始 |
| 冰川迷宫 | 滑溜地面，路径分叉，Boss 1 驻守 | 初始 |
| 暴风雪高地 | 强风持续吹偏移动方向，低温环境 | **高级雪服**（否则持续掉血）+平行式滑雪 |
| 地下冰窟 | 极暗区域，部分冰面需要高速通过，深处藏有卡宾技能 | **雪镜二级**（照明）+ **升级雪板**（高速通行） |
| 雪山顶峰 | 最终决战区，超强气流，地形极度复杂 | 全装备 + Boss 2 已击败 + **卡宾技能** |

## 三·五、游戏流程

### 开场
玩家扮演一名**滑雪骑士**，在雪山脚下醒来，装备只有一块普通雪板，目标是登上雪山顶峰，击败占领山顶的**雪山之王**。

### 阶段一：雪山入口（教学）
1. 学习基础滑行、跳跃、近战攻击
2. 遭遇第一批雪人兵，击败后获得初始经验
3. 在入口小屋找到**雪镜（一级）**，解锁攻击距离提升
4. 推进至冰川迷宫入口

### 阶段二：冰川迷宫
1. 地面打滑，需要掌握动量控制才能精准跳平台
2. 探索地图积累金币；在迷宫商人处**花 60 金币购买头盔**（解锁高速冲撞额外伤害）
3. **Boss 1：冰川雪豹** ── 高机动性，考验玩家利用斜坡速度反击
4. 击败后掉落**雪服碎片**（80 金币固定），前往铁匠处**花 100 金币合成高级雪服**；同时触发**进化 #1**，解锁**技能：平行式滑雪**（更快刹车、速度上限提升、可过更陡坡度，是进入暴风雪高地的必要条件）；等级上限从 10 级提升至 20 级

### 阶段三：暴风雪高地
- 入场门槛：穿戴**高级雪服**，否则每秒扣血（寒冷伤害）
1. 强侧风会持续推移角色，需要逆风斜滑抵消
2. 遭遇精英敌人：铠甲冰熊
3. 隐藏房间内存有**冰刃投掷技能**（宝箱），据点后战斗能力大幅提升
4. 找到**升级雪板图纸**，在工坊处**花 150 金币升级雪板**
5. **Boss 2：冻结守卫** ── 防御型，需要找破防时机，高速冲撞是关键
6. 击败后获得**雪镜二级升级零件**，再**花 80 金币**在商人处完成升级，解锁地下冰窟入口；触发**进化 #2**，解锁**技能：后空翻（二段跳）**；等级上限从 20 级提升至 30 级

### 阶段四：地下冰窟
- 入场门槛：**升级雪板**（部分极滑冰面需要高速才能通过）+ **雪镜二级**（深处极暗，无照明无法前进）
1. 迷宫结构最为复杂，墙面有隐藏通道
2. 深处决战**雪崩巨人（Boss 3）**，只有高速冲撞才能击退；击败后触发**进化 #3**，解锁**卡宾技能**，等级上限从 30 级提升至 40 级
3. 获得卡宾后，复杂的冰道路线全部开放，找到通往顶峰的缆车/冰道入口

### 阶段五：雪山顶峰（终局）
- 入场门槛：全套装备 + Boss 3 已击败 + **卡宾技能**（顶峰气流极强，地形复杂，无卡宾无法稳定操控）
1. 残酷强风区，需要频繁切换攻击和滑行节奏
2. 最后的精英守卫群战
3. **最终 Boss：雪山之王** ── 多阶段战斗，结合全部技能考验
4. 击败后，雪山解封，通关

### 回溯奖励
- 解锁新装备/技能后可回老区域，找到此前无法抵达的隐藏房间
- 隐藏内容：强化石（永久提升某项属性）、装备外观皮肤、剧情碎片

## 四、敌人设计

- **基础敌人**：雪人兵（近战）、冰猫（快速）、冰球炮台（远程）
- **精英敌人**：铠甲冰熊、雪崩巨人
- **Boss**：
  - Boss 1：冰川雪豹（机动 Boss，测试速度）
  - Boss 2：冻结守卫（防御型 Boss，测试攻击时机）
  - 最终 Boss：雪山之王

## 五、技术架构（Godot 4）

```
skiknight/
├── scenes/
│   ├── player/         # 玩家角色场景
│   ├── enemies/        # 敌人场景
│   ├── maps/           # 各地图场景

│   └── effects/        # 粒子特效
├── scripts/
│   ├── player/         # 移动、战斗、状态机
│   ├── enemies/        # AI 行为
│   ├── systems/        # 道具、存档、地图管理
│   └── ui/             # UI 逻辑
├── assets/
│   ├── sprites/
│   ├── sounds/
│   └── tilemaps/
└── autoload/           # 全局单例（GameManager、SaveSystem）
```

## 六、开发阶段规划

### Phase 1 — 原型验证
- 玩家基础移动（滑雪动量系统）
- 基础攻击
- 一张测试地图

### Phase 2 — 核心循环
- 敌人 AI（2-3 种）
- 成长系统（经验/升级）
- HUD（血量、技能栏）

### Phase 3 — 地图扩展
- 完整 5 个区域
- Metroidvania 锁钥设计
- 存档/读档系统

### Phase 4 — Boss & 打磨
- 3 个 Boss
- 特效、音效
- 平衡性调整
